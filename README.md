# WaysForGenuineEndpoint

## Create and Start Virtual Environment
### Windows
```sh
python -m venv myEnv

# Activate myEnv
myEnv\Scripts\activate
```
### Linux
```sh
python3 -m venv myEnv

# Activate myEnv
source myEnv/bin/activate
```

## Install Libraries
```sh
pip install -r requirements.txt
```

## Run Script
```sh
python EndpointsChecks.py
```
